## 📌 О проекте

**Random Task Generator** — это программа на Python, которая помогает генерировать случайные задачи для учёбы, спорта и работы. Программа сохраняет историю всех задач и позволяет добавлять свои собственные задачи.

### Основные возможности:
- 🎲 **Случайная генерация задачи** из готового списка
- ➕ **Добавление своей задачи** с выбором типа
- 🔍 **Фильтрация истории** по типу задачи (учёба/спорт/работа/все)
- 💾 **Сохранение истории** в JSON файл
- 📂 **Автозагрузка истории** при запуске программы
- ✅ **Проверка ввода** (нельзя добавить пустую или слишком длинную задачу)
- 📊 **Статистика** количества задач
