# Currency Converter — Конвертер валют

**Автор:** Баскаков Арсений Сергеевич

## Описание программы

GUI-приложение для конвертации валют с использованием актуальных курсов из открытого API. Позволяет конвертировать суммы между различными валютами, сохранять историю операций и управлять ею.

### Функциональность:
- **Конвертация валют** в реальном времени
- **Выбор из 24 популярных валют** (USD, EUR, RUB, GBP, JPY и др.)
- **История конвертаций** с датой и временем
- **Сохранение и загрузка** истории в JSON-файл
- **Очистка истории**
- **Автоматическое получение курсов** через API

## Как получить API-ключ

**ВАЖНО:** В данной реализации используется **бесплатный API без ключа** — `exchangerate-api.com` [citation:1][citation:5].

Однако, если вы хотите использовать собственный API-ключ для увеличения лимита запросов:

### Способ 1: ExchangeRate-API (рекомендуется)
1. Перейдите на сайт: https://www.exchangerate-api.com
2. Нажмите "Get Free API Key"
3. Зарегистрируйтесь (укажите email)
4. Получите API-ключ на почту
5. Бесплатный тариф: 1500 запросов в месяц, обновление раз в день [citation:2][citation:9]

### Способ 2: Open Exchange Rates API
1. Перейдите на https://openexchangerates.org
2. Зарегистрируйтесь
3. Бесплатный тариф: 1000 запросов в месяц
